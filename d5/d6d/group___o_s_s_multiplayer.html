Detailed Description

Game sessions, matchmaking, and multiplayer infrastructure.

Multiplayer interfaces provide session and matchmaking services:

  • Game session creation and management
  • Session browsing and search
  • Matchmaking (skill-based, rule-based)
  • Session V1 (legacy) and V2 (new) support
  • Dedicated server integration
  • Voice chat integration
